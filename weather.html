Project Overview

The Weather Forecast App is a web-based application that provides users with real-time weather information and forecasts for different locations. It retrieves weather data through APIs and displays details such as temperature, humidity, wind speed, and weather conditions in an intuitive interface. The project demonstrates API integration, data visualization, and responsive web development skills.
